# 业务专有知识库 (biz-knowledge/specific)

**业务项目专有, 项目隔离** (Q27 专有层)。

## 用途

- 业务专有术语 (项目内部术语)
- 业务专有数据 schema
- 业务专有 SOP (项目内具体 SOP)
- 业务专有最佳实践 (项目内沉淀)

## 写入规则

- **业务项目可写**: 业务项目自己管理自己目录
- **PMO 不干预**: PMO 只管契约, 不管内容
- **PMO 可验证**: PMO 实例可验证专有知识是否符合契约
- **版本管理**: Git 跟踪, 业务项目自管

## 目录结构

```
biz-knowledge/specific/
├── README.md (本文件)
├── <biz-project-id>/      # 每个业务项目独立子目录
│   ├── terms.md           # 业务专有术语
│   ├── data-schema.md     # 业务数据 schema
│   ├── sop/               # 业务专有 SOP
│   ├── best-practices/    # 业务专有最佳实践
│   └── README.md          # 业务项目知识库说明
```

## 业务项目自定 (Q11/Q12)

| 维度 | 业务项目 | PMO |
|---|---|---|
| 业务术语表 | **业务 agent 定义** | **PMO 验证** |
| 业务数据 schema | **业务项目定义** | **PMO 存** |

## 与业务通用知识库区别

| 维度 | 专有 (本目录) | 通用 (biz-knowledge/general/) |
|---|---|---|
| 访问 | 仅本项目 | 所有项目 |
| 修改 | 业务项目自定 | PMO 实例评审 |
| 用途 | 业务专有术语/schema/SOP | 通用术语/规范/模板 |
