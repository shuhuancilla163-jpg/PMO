## H3: 业务回滚粒度 (接口态)

业务回滚粒度由业务项目内部约定, PMO 只留接口:

| 回滚粒度 | 说明 | 业务项目自定 |
|---|---|---|
| 单事务回滚 | 单个业务操作回滚 | 业务项目填入 |
| 补偿事务 | 补偿逻辑回滚 | 业务项目填入 |
| 全量回滚 | 整个业务流回滚 | 业务项目填入 |

## H4: 业务流编排接口 (接口态)

业务流编排由业务项目内部定义, PMO 只留接口:

| 接口 | 说明 | 业务项目自定 |
|---|---|---|
| 编排入口 | 业务流启动方式 | 业务项目填入 |
| agent 顺序 | 业务 agent 执行顺序 | 业务项目填入 |
| 输入输出 | agent 之间数据传递 | 业务项目填入 |
| 触发条件 | 什么条件触发 | 业务项目填入 |
