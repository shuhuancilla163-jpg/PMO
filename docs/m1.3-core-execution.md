## PMO 规范不参与业务 (验证)

- **PMO 规范** (`immutable/0-governance/`) — 不包含业务具体内容
- **业务具体内容** — 业务项目自维护 (`biz-projects/`)
- **业务项目 2 层合规** — 业务项目整体 (m2.6) + 业务项目内研发 5 阶段
- **DEC-2026-0002 边界** — PMO 监管 ≠ PMO 干预
- **业务 agent 自定** — 业务项目按业务场景配置 (不固定 8 个)

## 关键 API

```python
from core_execution.core_execution import (
    PhaseFlowEngine, PhaseGateValidator, DecisionLog, ImmutableDocSigner,
    SponsorDashboard, PersonalityLoader, RoleIdentityLoader,
    ThreeDimensionAssessment, ThreeDimensionMonitor
)

# C1 阶段流转
pfe = PhaseFlowEngine("1.1", "/path/to/pmo")
pfe.advance(Phase.P2_ARCHITECTURE)

# C2 阶段门控
pgv = PhaseGateValidator("/path/to/pmo")
result = pgv.validate_gate("1.1", Phase.P5_SELF_TEST)

# C3 决策日志 (SQLite)
dl = DecisionLog("/path/to/pmo")
dl.add("DEC-2026-9999", "title", "category", "actor", True)

# C6 不可变文档签名
signer = ImmutableDocSigner("/path/to/pmo")
sig = signer.sign("immutable/0-governance/0.0.1-five-values.md")

# C9 Sponsor 面板
dash = SponsorDashboard("/path/to/pmo")
dashboard = dash.get_dashboard()

# C10 性格加载
pl = PersonalityLoader()
personality = pl.load("Sponsor")

# C15 身份加载
ril = RoleIdentityLoader()
identity = ril.load("L1_PMO-Main")

# DEC-2026-0002 3 维度考核
tda = ThreeDimensionAssessment(assessor_agent)
assessment = tda.assess(pmo_main, engineer_agent, monitor_agent, "1.1")

# DEC-2026-0002 3 维度监控
tdm = ThreeDimensionMonitor(pmo_main, engineer_agent, monitor_agent)
monitor = tdm.monitor_all("1.1")
```
