## 8 PMO 角色 (5 → 8, DEC-2026-0002)

### 三权分立

```
L0 监督权 (1 个)
└── Sponsor — 人机, 看指标看板, 不执行

L1 行政权 (1 个)
└── PMO-Main — 1 实例管 N 项目, 维度 1 业务项目整体采集

L2 司法权 (6 个, DEC-2026-0002 3 → 6)
├── Plan-Agent — 计划/治理
├── Engineer-Agent — 维度 2 业务项目内研发 5 阶段采集
├── Monitor-Agent — 维度 3 业务项目上报采集 + 上报合规监管
├── Reviewer-Agent — 审查/审计
├── Assessor-Agent — 3 维度分别考核
└── Message-Broker-Agent — 项目间消息经 PMO 中介
```

## 3 维度采集严格分离 (DEC-2026-0002)

| 角色 | 采什么 | 数据源 |
|---|---|---|
| **L1 PMO-Main** | 维度 1 (业务项目整体) | register.yaml + state + quota + archive + isolation |
| **L2 Engineer-Agent** | 维度 2 (研发 5 阶段) | eng-roles/ + 5 阶段产出 + 决策日志 + 不可变文档 |
| **L2 Monitor-Agent** | 维度 3 (业务项目上报) | 业务项目按 PMO 规范上报的关键指标 |

**关键**:
- 3 维度采集**不重叠**, 严格分离
- 1 个角色只采 1 个维度 (避免混乱)
- 1 个维度只被 1 个角色采 (避免重复)

## 三权分立 + 1 规范 N 项目

- **L0 Sponsor** 监督, 打破递归
- **L1 PMO-Main** 1 实例管 N 业务项目 (0.0.10)
- **L2 6 个司法权 agent** 审计 L1
- **三权分立**: 监督权 (L0) + 行政权 (L1) + 司法权 (L2)

## 演示

```python
from agents.agent_base import PMOInstance

# 1. 创建 PMO 实例
pmo = PMOInstance("/path/to/pmo")

# 2. 激活 8 角色
pmo.activate_all()
for agent in pmo.agents:
    print(f"{agent.layer} {agent.name}: {agent.reflect()['state']}")

# 3. L2 6 角色审计 L1
audits = pmo.l2_audit_l1()

# 4. 3 维度采集
pmo.pmo_main.process({"action": "register_project", "project_id": "1.1"})
pmo.engineer_agent.process({"action": "collect_eng_stage_data", "project_id": "1.1", "stage": "requirement", "data": {}})
pmo.monitor_agent.process({"action": "collect_reported_data", "project_id": "1.1", "data": {...}})

# 5. Assessor 3 维度考核
assessment = pmo.assessor_agent.assess_project(pmo.pmo_main, pmo.engineer_agent, pmo.monitor_agent, "1.1")

# 6. Message-Broker 业务↔业务
pmo.message_broker_agent.process({"action": "publish", "from_project": "1.1", "topic": "biz.1.2.state", "message": {}})

# 7. L0 Sponsor 监督
observation = pmo.sponsor_agent.observe_metrics(pmo.pmo_main)
```
